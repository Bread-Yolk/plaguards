Tutorial

1. Open tools menu or "Try Plaguards" button in home page.

2a. For IOC Checker Feature, on the search bar, input parameters that are server needed (ex: hash <value>, signature <value>, domain <value>, ip <value>).

Note: Please add space between the initial parameter and the value.

2b. For Deobfuscation Feature, on the upload section, upload the file code that you want to analyze in .ps1 or .txt format.

3. Click the search (for IOC Checker) or file upload menu (for Deobfusaction feature) and wait for the results.

4. Open the PDF Report

5a. For IOC Checker Feature, You will see your IoC data if exist.

5b. For Deobfuscation Feature, you will see your code deobfuscation result with domain and IP IoC data if exist.
