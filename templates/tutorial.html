Tutorial

1. Open tools menu or "Try Plaguards" button in home page.

2. For IOC Checker Feature, on the search bar, input parameters that are server needed (ex: hash <value>, signature <value>, domain <value>, ip <value>).

Note: Please add space between the initial parameter and the value.

3. Click the search (for IOC Checker) or file upload menu (for Deobfuscation feature) and wait for the results.

4. Open the PDF Report. Please note that the report will be available for viewing for only 5 minutes on the server, so We suggest you to download it.

5. For IOC Checker Feature, You will see your IoC data if exist.

If you still do not understand the instructions above, please watch the video below.
